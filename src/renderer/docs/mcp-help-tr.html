MCP & Plugin Kurulum Rehberi

AI Tool'daki her entegrasyonu nasıl kurar ve yetkilendirirsiniz.

Figma Eklenti

Tasarım dosyaları, bileşenler, stiller.

  1. Uygulamada "Claude'a Kur" butonuna tıklayın.
  2. Claude Code'u açıp /mcp yazın.
  3. figma seçip "Authenticate" tıklayın.
  4. Tarayıcıda Figma hesabınızla giriş yapıp yetkilendirin.
API key gerekmez. Figma OAuth kullanır — sorulduğunda giriş yapmanız yeterli.

Jira (Atlassian) MCP

Issue takibi, sprint yönetimi.

  1. Uygulamada "Claude'a Kur" butonuna tıklayın.
  2. Claude Code'u açıp /mcp yazın.
  3. atlassian seçip "Authenticate" tıklayın.
  4. Atlassian hesabınızla giriş yapıp erişim verin.
Atlassian, uzaktaki bir HTTP MCP sunucusu üzerinden OAuth kullanır. Uygulama Atlassian MCP kaydını tüm projelerde geçerli olacak user-level Claude MCP yapılandırmasına yazar; ardından /mcp içinden yetkilendirirsiniz. Jira projelerinizde Browse, Create, Edit, Delete ve Transition yetkileriniz olduğundan emin olun.

PostgreSQL MCP

Veritabanı sorgusu, şema inceleme.

  1. Uygulamada "Claude'a Kur" butonuna tıklayın.
  2. Bağlantı formunu doldurun: Host, Port, Veritabanı, Kullanıcı Adı, Şifre.
  3. Modaldaki "Claude'a Kur" butonuna tıklayın.
  4. Uygulama kaydetmeden önce bağlantıyı otomatik olarak doğrular.
PostgreSQL, manuel kimlik bilgisi gerektiren tek entegrasyondur. Bağlantı bilgileri user-level Claude MCP yapılandırması olan ~/.claude.json dosyasında saklanır.
