MCP & Plugin Setup Guide

How to set up and authenticate each integration in AI Tool.

Figma Plugin

Design files, components, styles.

  1. Click "Install to Claude" in the app.
  2. Open Claude Code and run /mcp.
  3. Select figma and click "Authenticate".
  4. Sign in with your Figma account in the browser and authorize.
No API key needed. Figma uses OAuth — just sign in when prompted.

Postman MCP

API collections, testing.

  1. Click "Install to Claude" in the app.
  2. Open Claude Code and run /mcp.
  3. Select postman and click "Authenticate".
  4. Sign in with your Postman account in the browser and authorize.
Postman MCP uses OAuth. No manual API key required.

Jira (Atlassian) MCP

Issue tracking, sprint management.

  1. Click "Install to Claude" in the app.
  2. Open Claude Code and run /mcp.
  3. Select atlassian and click "Authenticate".
  4. Sign in with your Atlassian account and grant access.
Uses Atlassian OAuth through a remote HTTP MCP server. The app writes the Atlassian MCP entry into your user-level Claude MCP config so it is available across projects, then you authenticate from /mcp. Make sure you have Browse, Create, Edit, Delete, and Transition permissions on your Jira projects.

GitHub Plugin

Pull requests, issues, code review.

  1. Click "Install to Claude" in the app.
  2. Open Claude Code and run /mcp.
  3. Select github and click "Authenticate".
  4. Sign in with your GitHub account and authorize the app.
GitHub uses OAuth. No personal access token required.

PostgreSQL MCP

Database queries, schema inspection.

  1. Click "Install to Claude" in the app.
  2. Fill in the connection form: Host, Port, Database, Username, Password.
  3. Click "Install to Claude" in the modal.
  4. The app validates the connection automatically before it saves the configuration.
PostgreSQL is the only integration that requires manual credentials. The connection string is stored in your user-level Claude MCP config at ~/.claude.json.

Other Plugins

Superpowers, Playwright, TypeScript LSP, Code Review, Security Guidance, Frontend Design, Code Simplifier

  1. Click "Install to Claude" in the app.
  2. Done! These plugins work immediately — no authentication needed.
These plugins don't require any authentication or API keys. They are ready to use right after installation.
